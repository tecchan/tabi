
# タビノタネ

**自分が知っている場所しか目的地が思い浮かばない**  
**旅行に行きたい人**向けの  
**タビノタネ**という**目的地レコメンドサービス**です。  

これは、**シンプルな質問に答えると目的地がレコメンドされ**、   
**楽天トラベルのようなWebサービス**とは違って、   
**思いつかないような場所を教えてくれる機能**が備わっています。
